# Export a report to Excel

**Goal:** download a report as an `.xlsx` workbook to share or archive.

## Steps

### A single report

1. Open a finished job's **Results** page and select the tab you want
   (**Vulnerabilities**, **Licenses**, or **Version Currency**).
2. Click **Export to Excel**.
3. Your browser downloads an `.xlsx` file containing every row of that report (the full
   report, not just the filtered view).

### Everything at once

1. Select the **Overview** tab.
2. Click **Export all to Excel**.
3. You get one workbook with a separate sheet per report.

## Result

An Excel workbook is saved locally. Links are preserved where they appear — for example,
in the Version Currency sheet the package name is a clickable link to its registry page.
The export is generated in your browser, so no data leaves your machine to produce it.
See also the [User Guide](../user-guide/index.md).
